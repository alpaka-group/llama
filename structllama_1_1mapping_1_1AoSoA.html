template<typename TArrayExtents, typename TRecordDim, std::size_t Lanes, typename TLinearizeArrayDimsFunctor = LinearizeArrayDimsCpp, template< typename > typename FlattenRecordDim = FlattenRecordDimInOrder>
struct llama::mapping::AoSoA< TArrayExtents, TRecordDim, Lanes, TLinearizeArrayDimsFunctor, FlattenRecordDim >

Array of struct of arrays mapping. Used to create a View via allocView.

Template Parameters
LanesThe size of the inner arrays of this array of struct of arrays.
FlattenRecordDimDefines how the record dimension's fields should be flattened. See FlattenRecordDimInOrder, FlattenRecordDimIncreasingAlignment, FlattenRecordDimDecreasingAlignment and FlattenRecordDimMinimizePadding.
