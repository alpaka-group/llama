Detailed Description

template<typename T_Mapping, typename BlobType>
struct llama::View< T_Mapping, BlobType >

Central LLAMA class holding memory for storage and giving access to values stored there defined by a mapping. A view should be created using allocView.

Template Parameters
T_MappingThe mapping used by the view to map accesses into memory.
BlobTypeThe storage type used by the view holding memory.
