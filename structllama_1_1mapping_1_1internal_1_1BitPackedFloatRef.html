Detailed Description

template<typename Float, typename StoredIntegralPointer, typename VHExp, typename VHMan, typename SizeType>
struct llama::mapping::internal::BitPackedFloatRef< Float, StoredIntegralPointer, VHExp, VHMan, SizeType >

A proxy type representing a reference to a reduced precision floating-point value, stored in a buffer at a specified bit offset.

Template Parameters
FloatFloating-point data type which can be loaded and store through this reference.
