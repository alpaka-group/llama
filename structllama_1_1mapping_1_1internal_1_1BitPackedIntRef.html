Detailed Description

template<typename Integral, typename StoredIntegralPointer, typename VHBits>
struct llama::mapping::internal::BitPackedIntRef< Integral, StoredIntegralPointer, VHBits >

A proxy type representing a reference to a reduced precision integral value, stored in a buffer at a specified bit offset.

Template Parameters
IntegralIntegral data type which can be loaded and store through this reference.
StoredIntegralPointerPointer to integral type used for storing the bits.
