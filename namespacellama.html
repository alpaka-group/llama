) -> View<Mapping, internal::AllocatorBlobType<Allocator>>
inline

Creates a view based on the given mapping, e.g. AoS or :SoA. For allocating the view's underlying memory, the specified allocator callable is used (or the default one, which is bloballoc::Vector). The allocator callable is called with the size of bytes to allocate for each blob of the mapping. This function is the preferred way to create a View.
