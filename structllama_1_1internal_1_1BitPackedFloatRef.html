Detailed Description

template<typename Float, typename StoredIntegralPointer>
struct llama::internal::BitPackedFloatRef< Float, StoredIntegralPointer >

A proxy type representing a reference to a reduced precision floating-point value, stored in a buffer at a specified bit offset.

Template Parameters
IntegralIntegral data type which can be loaded and store through this reference.
StoredIntegralPointerPointer to integral type used for storing the bits.
