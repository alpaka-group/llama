Detailed Description

template<typename Mapping, typename CountType = std::size_t, bool MyCodeHandlesProxyReferences = true>
struct llama::mapping::Trace< Mapping, CountType, MyCodeHandlesProxyReferences >

Forwards all calls to the inner mapping. Traces all accesses made through this mapping and allows printing a summary. /tparam Mapping The type of the inner mapping. /tparam CountType The type used for counting the number of accesses. /tparam MyCodeHandlesProxyReferences If false, Trace will avoid proxy references but can then only count the number of address computations
