Detailed Description

template<typename Mapping, typename TCountType = std::size_t, bool MyCodeHandlesProxyReferences = true>
struct llama::mapping::Trace< Mapping, TCountType, MyCodeHandlesProxyReferences >

Forwards all calls to the inner mapping. Traces all accesses made through this mapping and allows printing a summary.

Template Parameters
MappingThe type of the inner mapping.
TCountTypeThe type used for counting the number of accesses.
MyCodeHandlesProxyReferencesIf false, Trace will avoid proxy references but can then only count the number of address computations
