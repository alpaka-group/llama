Detailed Description

template<typename T_ArrayDims, typename T_RecordDim, bool SeparateBuffers = true, typename T_LinearizeArrayDimsFunctor = LinearizeArrayDimsCpp>
struct llama::mapping::SoA< T_ArrayDims, T_RecordDim, SeparateBuffers, T_LinearizeArrayDimsFunctor >

Struct of array mapping. Used to create a View via allocView.

Template Parameters
SeparateBuffersIf true, every element of the record dimension is mapped to its own buffer.
LinearizeArrayDimsFunctorDefines how the array dimensions should be mapped into linear numbers and how big the linear domain gets.
